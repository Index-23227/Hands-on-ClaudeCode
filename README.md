# Hands-on ClaudeCode

재경팀 업무 자동화를 주제로 한 Claude Code 실습 모음. 주차별로 난이도를 올려가며 AI에게 업무를 시키는 법을 익힙니다.

---

## 주차별 실습

| 주차 | 주제 | 링크 |
|------|------|------|
| Week 04 | CLAUDE.md 만들고 바이브코딩 체험 | [week04-hands-on/](week04-hands-on/README.md) |
| Week 05 | Flask로 매출 데이터 시각화 | [week05-hands-on/](week05-hands-on/README.md) |
| Week 06 | DART/환율 API 연동 파이프라인 | [week06-hands-on/](week06-hands-on/README.md) |
| Week 07 | "이런 것도 됩니다" 바이브코딩 쇼케이스 | [week07-hands-on/](week07-hands-on/README.md) |

---

## 폴더 구조

```
Hands-on-ClaudeCode/
├── week04-hands-on/   ← CLAUDE.md 하네스 입문
├── week05-hands-on/   ← DB + Flask 대시보드
├── week06-hands-on/   ← 외부 API 파이프라인
├── week07-hands-on/   ← 강사 시연용 데모
└── setting/           ← 참고용 사업보고서 PDF
```

각 주차 폴더 안에 `README.md`(실습 가이드), `CLAUDE.md`(하네스), `exercises/`(단계별 안내), `answers/`(예시 답안), `data/`(실습 데이터)가 들어있습니다.

---

## 시작하기

1. VSCode로 이 저장소를 엽니다.
2. 진행할 주차 폴더를 열고 해당 폴더의 `README.md`를 먼저 읽습니다.
3. `exercises/` 안내를 따라 Claude Code에게 작업을 시킵니다.
4. 막히면 `answers/`의 예시 답안을 참고합니다.
