# 막힘 카드 — 이럴 땐 이렇게

> 막혔다고 손 들기 전에 이 카드 먼저 보세요.
> 70%는 여기서 풀립니다. 안 풀리면 강사 호출.

---

## 1. 빨간 글씨(에러)가 떴어요

**하지 말 것**: 얼어붙지 마세요. 에러는 Claude가 읽으면 대부분 풉니다.

**이렇게 하세요**:
1. 에러 메시지를 **처음부터 끝까지** 드래그해서 복사 (Traceback 전체)
2. Claude에게 그대로 붙여넣고 이 한 줄:

```
위 에러 해결해줘. 원인이 뭐고 어떻게 고치면 돼?
```

3. Claude가 수정안을 주면 "그럼 그렇게 고쳐서 다시 실행해줘"

**그래도 안 되면**: 같은 에러가 3번 반복되면 강사 호출.

## 2. 내가 하고 싶은 게 어떤 유형인지 모르겠어요

아래 중에 가까운 걸 찾아서 `demos/` 폴더의 **데모 프롬프트를 복사해서 내 상황으로 바꿔 쓰세요.**

| 내 상황 | 가까운 데모 | 프롬프트 힌트 |
|---|---|---|
| 두 엑셀 파일을 비교하고 싶음 | 데모 1 | "A.xlsx와 B.xlsx를 비교해서 다른 행만 뽑아줘" |
| PDF/문서에서 숫자·항목 뽑고 싶음 | 데모 2 | "이 PDF에서 ○○, ○○, ○○를 뽑아 엑셀로 정리해줘" |
| DB/엑셀에 질문하고 답 받고 싶음 | 데모 3 | "○○에서 ○○별 ○○ 합계를 알려줘" (SQL 몰라도 됨) |
| 보고서 서식 꾸미고 싶음 (색·차트·합계) | 데모 4 | "○○ 데이터로 보고서 만들어줘. 헤더는 남색, 합계는 볼드..." |
| 두 시스템 데이터가 맞는지 대조 | 데모 5 | "A와 B를 비교해서 안 맞는 행만 빨간 배경으로 표시" |
| 매달 같은 양식에 숫자만 바뀜 | 데모 6 | "○○ 데이터로 4월·5월·6월 보고서를 한번에 만들어줘" |

## 3. Claude가 엉뚱한 걸 만들었어요

**처음부터 다시 시키지 마세요.** 90% 맞을 땐 나머지 10%만 고치면 됩니다.

**이렇게 말하세요**:
```
거의 맞는데 ○○ 부분만 ○○로 바꿔줘. 나머지는 그대로.
```

예시:
- "거의 맞는데 금액 컬럼 숫자에 천단위 콤마만 넣어줘"
- "좋은데 합계 행이 빠졌어. 맨 아래에 합계 넣어줘"
- "색깔 반대야. 증가를 빨강, 감소를 초록으로 바꿔줘"

---

## 4. 엑셀 구조가 복잡해서 Claude가 못 읽어요

병합셀·색상·수식이 많으면 Claude가 "데이터가 어딘지 모르겠다" 상태가 됩니다.

**이렇게 시작하세요**:
```
이 엑셀의 구조부터 알려줘. 어떤 시트에 어떤 컬럼이 어디부터 있어?
```

구조를 파악한 다음 원래 하려던 작업을 시키세요.

---

## 5. 내 데이터를 그대로 올려도 되나 고민돼요

**민감정보(실명·실계좌·실주민번호 등)는 올리지 마세요.**

**이렇게 하세요**:
```
이 엑셀의 법인명·계좌번호·금액 같은 민감정보를 가상 데이터로 바꿔줘.
구조는 유지한 채로 익명화해서 저장해줘.
```

익명화된 파일로 실습하세요.
