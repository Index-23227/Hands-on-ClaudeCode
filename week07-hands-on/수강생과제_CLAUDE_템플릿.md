# 내 업무 자동화 CLAUDE.md — 수강생용 템플릿

> 이 파일을 **본인 업무 폴더에 `CLAUDE.md`로 복사**해서 빈칸을 채우세요.
> 직접 타이핑하기 어려우면 **Claude에게 말로 설명**하고 채워달라고 시키세요:
>
> ```
> 내 업무 CLAUDE.md를 채우려고 해. 내가 설명할 테니 아래 템플릿의
> 빈칸을 채워줘. [이 파일 내용 복붙] 모르는 항목은 나한테 물어봐줘.
> ```
>
> 빈칸이 다 채워지면 그때부터 실제 자동화 요청을 시작합니다.

## 1. 현재 어떻게 하고 있나 (손으로 하는 과정)

지금 이 일을 **손으로 어떻게 하고 있는지** 단계별로 적어주세요. Claude가 자동화하려면 먼저 "사람이 뭘 하는지"를 알아야 합니다.

1. _(예: 이메일로 8개 법인에서 엑셀을 받는다)_
2. _(예: 모든 파일을 한 폴더에 모은다)_
3. _(예: 새 엑셀을 열어서 8개 파일의 매출 컬럼을 복사·붙여넣는다)_
4. _(예: 통화가 다른 금액은 환율 찾아서 원화로 환산한다)_
5. _(예: 합계 구하고 서식 정리해서 상사에게 보낸다)_

매 단계에 걸리는 시간도 대략 적어두면 Claude가 우선순위를 잡기 좋습니다.
