# Step 1 — 본인 업무 가져오기 워크숍 (90분)

> 지금부터는 **여러분이 주인공**입니다.
> 본인 업무 중 하나를 골라 Claude와 함께 자동화합니다.
> 오늘 안에 전부 끝낼 필요 없습니다. **첫 한 단계**만.

---


## 1단계: 설계 — CLAUDE.md 채우기 (15분)

### 1-1. 본인 업무 폴더 만들기

원하는 위치(예: `바탕화면/내업무자동화/`)에 폴더 하나 만드세요.
Claude에게 시켜도 됩니다:

```
바탕화면에 "내업무자동화" 폴더를 만들어줘.
```

### 1-2. 템플릿 복사

`week07-hands-on/수강생과제_CLAUDE_템플릿.md`를 열어서
**전체 내용을 복사 → 방금 만든 폴더에 `CLAUDE.md`로 저장**.

Claude에게 시키면 더 쉬움:

```
week07-hands-on/수강생과제_CLAUDE_템플릿.md 내용을 그대로 복사해서
바탕화면/내업무자동화/ 폴더에 CLAUDE.md로 저장해줘.
```

### 1-3. 빈칸 채우기 — Claude에게 맡기세요

직접 타이핑하지 마세요. **Claude에게 말로 설명**하고 채워달라고 시킵니다:

```
내가 자동화하려는 업무를 설명할 테니 방금 만든 CLAUDE.md의
빈칸을 채워줘. 모르는 항목은 나한테 하나씩 물어봐줘.

내 업무는: [한 문장으로 설명]
지금 내가 손으로 하는 과정은: [단계별로 말로 설명]
가지고 있는 파일은: [엑셀인지, 몇 개인지, 어디 있는지]
```

Claude가 부족한 정보를 물어보면 답하세요. 15분 안에 **8번 "오늘 목표"까지** 채우세요.

### 🚨 민감정보 체크

실명·계좌번호·실제 거래처명이 데이터에 있으면 **Claude에 올리기 전에** 익명화하세요:

```
이 엑셀에서 법인명·담당자명·계좌번호를 가상 데이터로 바꿔줘.
구조는 유지한 채로. 익명화된 파일을 같은 폴더에 "익명_원본이름.xlsx"로 저장해줘.
```

---

## 2단계: 구현 — Claude에게 요청 (60분)

### 2-1. 첫 요청

CLAUDE.md 8번 "오늘 목표"에 적은 내용을 Claude에게 시킵니다:

```
방금 채운 CLAUDE.md 규칙에 따라서 [8번 목표]를 만들어줘.
```

### 2-2. Claude가 만들면 바로 실행

Claude가 코드를 만든 다음 **"이거 실행해줘"**라고 한 번 더 말하세요.
결과 파일이 생성되면 다음 단계로.

### 2-3. 결과가 이상하면 — 수정 프롬프트 패턴

처음부터 다시 시키지 마세요. 쇼케이스에서 본 패턴 그대로:

```
거의 맞는데 [이 부분]만 [이렇게] 바꿔줘. 나머지는 그대로.
```

예:
- "거의 맞는데 합계 행이 빠졌어. 맨 아래에 넣어줘."
- "열 순서가 달라. 법인코드가 제일 왼쪽, 그다음 법인명 순서로 바꿔줘."
- "색깔이 반대야. 증가를 빨강, 감소를 초록으로."

### 2-4. 막히면 — 막힘카드

5분 이상 못 풀면 [../막힘카드.md](../막힘카드.md) 펴세요. 대부분 여기서 풀립니다:
- 에러 났으면 → 에러 통째로 Claude에 붙여넣기
- 뭘 만들지 막연하면 → 6개 데모 중 가까운 것 찾아 프롬프트 복붙
- 그래도 3번 반복 실패 → 강사 호출

---

## 3단계: 검증 — 신뢰 가능한가? (15분)

### 3-1. 검증체크리스트 돌리기

[../검증체크리스트.md](../검증체크리스트.md) 5개 항목 **전부 통과**시켜야 실제 업무에 쓸 수 있습니다.

| 항목 | 확인 |
|---|---|
| ① 행 개수 맞나 | ☐ |
| ② 합계 맞나 | ☐ |
| ③ 샘플 1~2건 수기 대조 | ☐ |
| ④ 포맷 (숫자·날짜) | ☐ |
| ⑤ 예외 케이스 (빈 값·음수 등) | ☐ |

### 3-2. 실패하면

해당 항목을 짚어서 Claude에게:

```
합계가 내가 계산한 거랑 달라. 내가 계산한 건 ○원이고
결과는 ○원이야. 어디서 차이가 났는지 찾고 고쳐줘.
```

수정 후 **체크리스트를 처음부터 다시** 돌립니다.

### 3-3. 여전히 막히면 → 스코프를 줄이세요

완벽 추구 금지. 오늘 80% 맞으면 충분합니다. 나머지 20%는 숙제.

## 마무리 (15분, 강사 주도)

1~2명이 본인 결과물을 공유합니다. 강사가 질문:
- 어디에서 가장 많이 막혔나?
- 처음 CLAUDE.md를 채울 때 어려웠던 것은?
- Claude가 틀렸을 때 어떻게 고쳐 말했나?

**8주차 예고**: 오늘 만든 것을 완성해서 가져오는 것이 8주차 결과물입니다.
미완성도 괜찮습니다. 오늘의 90%는 **Claude와 대화하는 감각**이고, 완성본은 다음 주 숙제.
