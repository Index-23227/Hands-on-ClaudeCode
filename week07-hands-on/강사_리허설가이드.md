# 강사 리허설 가이드 — 수업 전 드라이런 (5데모 신구조)

> **수강생 2명(김인혜·육아름) 맞춤 쇼케이스 5종**을 실제로 한 번 돌려봐야 당일 당황 안 함.
> 이 문서는 그 리허설을 **90~100분**에 끝낼 수 있게 구조화한다.
> 리허설에서 막힌 지점 = 수업에서 수강생이 헷갈릴 지점.

---

## 소요 시간

- **빠른 버전**: 60분 (환경 10 + 데모 리허설 40 + 복기 10)
- **충실한 버전**: 100분 (환경 15 + 데모 리허설 70 + 복기 15)

첫 리허설은 충실한 버전 권장.

---

## 준비물

- [ ] Python 3.10+ + `openpyxl`·`python-pptx`·`Pillow` 설치
- [ ] `prep/.env` 파일 (GMAIL_USER, GMAIL_APP_PASSWORD, SUB_ACCOUNTS, DART_API_KEY)
- [ ] Gmail 2FA + 앱 비밀번호 발급
- [ ] 강사 부계정 2개 (`SUB_ACCOUNTS`에 넣음)
- [ ] MCP Gmail 서버 설정 (또는 IMAP 폴백 쓸 결심)
- [ ] DART API 키 (week06/.env에 이미 있으면 자동 탐색됨)
