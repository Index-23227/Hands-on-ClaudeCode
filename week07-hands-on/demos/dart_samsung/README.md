> **수강생이 느낄 것**: "Claude가 인터넷까지 나가서 공시 데이터를 가져와 해석까지 해주네"
> **핵심 메시지**: Week 4~6은 전부 로컬 파일 기반이었다. 이번엔 **실시간 외부 데이터 + AI 요약** 체인.
> **시연 시간**: 12분

## 시연 흐름

### 사전 — DART API 키 확인
`.env`에 `DART_API_KEY` 있어야 라이브 호출 가능. 없으면 폴백 엑셀 사용.

발급: https://opendart.fss.or.kr/uss/umt/EgovMberInsertView.do (무료, 즉시 발급)

### 프롬프트 (그대로 복붙)

```
금융감독원 DART(opendart.fss.or.kr)에서 삼성전자(주) 최근 2개 사업연도
연결재무제표를 가져와줘.

- corp_code: 00126380 (삼성전자)
- 엔드포인트: https://opendart.fss.or.kr/api/fnlttSinglAcntAll.json
- 파라미터: crtfc_key={.env의 DART_API_KEY}, bsns_year=2024 / 2025,
  reprt_code=11011 (사업보고서), fs_div=CFS (연결)
- 가져오는 것: 매출액, 영업이익, 당기순이익

가져온 후:
1. 2024 vs 2025 비교표 만들기 (증감액·증감률 포함)
2. 영업이익률·순이익률도 계산해서 추가
3. 엑셀로 저장: week07-hands-on/demos/dart_samsung/output/삼성전자_재무비교.xlsx
   - 헤더 짙은 남색 배경 + 흰 글씨
   - 증가는 빨강, 감소는 초록 (한국식)
   - 숫자는 조·억 단위로 보기 좋게 (예: 258,935,000,000,000원 → 258.9조원)
4. 엑셀 하단에 "이번 실적에서 주목할 포인트 3줄" 자연어 요약 셀로 추가
```

### 라이브 중 일어날 일

1. Claude가 `WebFetch`로 DART API 호출
2. JSON 응답 파싱 → 수백 개 계정 중 주요 3개만 추출
3. 비교표 계산 (증감률 등)
4. openpyxl로 엑셀 생성
5. 최하단에 자연어 요약 (Claude가 숫자 보고 직접 작성)

### 기대 결과

| 항목 | 2024 | 2025 | 증감액 | 증감률 |
|---|---|---|---|---|
| 매출액 | 258.9조 | 290.0조 | +31.1조 | **+12.0%** |
| 영업이익 | 32.7조 | 42.3조 | +9.6조 | **+29.4%** |
| 당기순이익 | 28.5조 | 35.4조 | +6.9조 | **+24.2%** |
| 영업이익률 | 12.6% | 14.6% | +2.0%p | |
| 순이익률 | 11.0% | 12.2% | +1.2%p | |

> 값은 대략치. 실제 공시일 기준 값으로 갱신됨.

DART API가 느리거나·장애나·키 없음 → 즉시 폴백:

1. `.env`에 `DART_API_KEY` 있으면 실제 호출 시도
