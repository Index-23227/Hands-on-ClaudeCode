# Case 3 — 개인화된 이메일 일괄 생성

> **난이도 ★★☆ — 재경팀 연관성 ★★☆**
> "반복 작업 한 번에 끝낸다"의 교과서적 사례. 연락처 엑셀로 N개의 개인화된 메일 초안 생성.

---

## 하려는 일

연락처 엑셀에 담긴 10~20명에게 **각자 상황에 맞춰 문구가 바뀌는** 이메일 초안을 한 번에 생성합니다.

### 완성 기준

- 수신자별 메일 1개 = 1개 텍스트 파일 or 1개 엑셀 행
- 본문에 수신자 이름·회사명·지난 거래 금액·마감일이 자동으로 들어감
- 미납자에게는 다른 톤(부드럽게 독촉)
- 결과 저장: `메일초안/{수신자}.txt` 또는 `메일초안_일괄.xlsx`

---

## 샘플 데이터

Claude에게 샘플 연락처를 만들라고 합니다:

```
가상의 거래처 연락처 엑셀을 만들어줘 (15건).
컬럼: 이름, 회사명, 이메일, 지난거래금액(원), 마감일, 상태(정상/미납).
미납 건이 3건 섞이게. 회사명·이름은 모두 가상.
내 업무 폴더에 "거래처_연락처.xlsx"로 저장.
```

---

## Step 1: CLAUDE.md 빈칸 채우기 (5분)

### 미리 알려진 값

- **0번**: 거래처 15명에게 개인화된 이메일 초안 일괄 생성
- **2-4번 컬럼**: 이름 / 회사명 / 이메일 / 지난거래금액 / 마감일 / 상태
- **3-1번 결과 형식**:
  - 선택 A: 텍스트 파일 15개 (수신자별)
  - 선택 B: 엑셀 1개 (수신자 1행 × 제목·본문 컬럼)
  - 중에서 하나 선택
- **3-3번 메일에 들어갈 요소**:
  - 인사: "안녕하세요, {회사명} {이름}님"
  - 거래 내역: "지난달 거래하신 {금액}원 관련하여..."
  - 마감 안내: "마감일은 {마감일}입니다"
  - 서명: 본인 이름 + 회사
- **4-1번 톤 규칙**:
  - 상태=정상: 정중한 안내
  - 상태=미납: 부드러운 독촉 (공격적이지 않게)
- **4-2번 예외**:
  - 이메일 주소가 빈 칸이면 건너뛰고 경고 로그
  - 지난거래금액이 0 또는 음수면 다른 템플릿 (단순 안부)
- **5-1번 민감정보**: 샘플 가상 데이터만 사용. 실제 거래처 쓰지 말 것.
- **6번 검증**:
  - 개수: 상태=정상/미납 건수와 생성된 정상/미납 톤 건수 일치
  - 치환: 샘플 1개 열어서 이름·회사·금액이 제대로 들어갔는지
- **8번 오늘 목표**: 15명분 메일 초안 1회 생성

---

## Step 2: Claude에게 요청 (20분)

```
방금 채운 CLAUDE.md 규칙에 따라 거래처_연락처.xlsx의 15명에게
개인화된 이메일 초안을 일괄 생성해줘. 출력은 엑셀 1개로.
```

### 흔히 나오는 문제

- **모든 메일이 똑같은 톤**
  ```
  "상태" 컬럼을 보고 정상은 정중 안내, 미납은 부드러운 독촉으로
  톤을 확실히 구분해줘. 지금은 다 비슷해 보여.
  ```
- **치환 실패 — 플레이스홀더가 그대로 남음**
  ```
  본문에 {이름}, {회사명} 같은 플레이스홀더가 치환 안 되고 그대로 있어.
  실제 값으로 다 채워줘.
  ```
- **금액 포맷이 이상함**
  ```
  금액에 천단위 콤마와 "원" 붙여줘 (예: "1,234,500원").
  ```
- **이메일 주소 빈 칸 무시 없이 에러**
  ```
  이메일 주소가 빈 칸이면 그 행은 건너뛰고 "누락: {이름}" 로그만 남겨줘.
  전체 중단하지 말고.
  ```

## Step 3: 검증 (10분)

### 이 케이스만의 검증 포인트

1. **샘플 1개 수신자 메일을 처음부터 끝까지 읽기** — 자연스럽게 읽히나, 이름·회사·금액이 제대로 들어갔나
2. **미납자 메일 1개 열기** — 정상 메일과 톤이 명확히 다른가
3. **빠진 수신자 없나** — 원본 15명 ↔ 결과 15건 (또는 이메일 누락 기록)
4. **금액·마감일 포맷** — 1,234,500원 / 2026-04-30 형태

---

## 확장 (시간 남으면)

- 상사 이름·회사 서명 자동 삽입
- Outlook으로 바로 보낼 수 있게 `.msg` 형식 생성 (추가 라이브러리 필요)
- 회신이 오는 경우를 대비한 FAQ 응답 템플릿 함께 생성

## 왜 이 케이스가 강력한가

비재경팀 수강생도 공감하기 쉽습니다. "거래처 20곳에 일일이 메일 쓰느라 반나절" 경험이 있는 사람이라면 이 데모의 가치를 30초 안에 이해합니다.
재경팀에게는 **미수금 독촉장 일괄 발송**, **환급 안내 통지**, **연말정산 안내** 등으로 그대로 이어집니다.
