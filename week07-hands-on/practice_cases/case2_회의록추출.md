# Case 2 — 회의록에서 액션아이템 추출

> **난이도 ★☆☆ — 재경팀 연관성 ★☆☆**
> 가장 쉬운 케이스. 텍스트 파일에서 "누가 뭘 언제까지 해야 하는지"를 뽑아 엑셀로 정리.

---

## 하려는 일

장문의 회의록 텍스트에서 **액션아이템(할 일)**을 찾아 엑셀 표로 만듭니다.

### 완성 기준

- 액션아이템별 1행
- 컬럼: 담당자 / 할 일 / 마감일 / 우선순위 / 비고
- 마감 지난 항목은 **빨간 글씨**
- 담당자 없는 항목은 "미정"으로 표시
- 결과 파일명: `회의록_액션아이템_YYYY-MM-DD.xlsx`

---

## 샘플 데이터

Claude에게 샘플 회의록을 만들어달라고 합니다:

```
재경팀 월례 회의록 샘플을 텍스트로 만들어줘 (800자 정도).
회의 일시 2026-04-15. 참석자 팀장·김대리·이주임·박사원.
자연스러운 대화 스타일이되, 액션아이템이 명시적·암묵적으로
5~7개 섞여있게. 일부는 담당자 명시, 일부는 "다음 주까지" 같은
막연한 기한. 샘플 파일을 내 업무 폴더의 "회의록_2026-04-15.txt"로 저장해줘.
```

---

## Step 1: CLAUDE.md 빈칸 채우기 (5분)

### 미리 알려진 값

- **0번**: 회의록 텍스트에서 액션아이템만 뽑아 엑셀로 정리
- **2-1번 파일 형식**: 텍스트(`.txt`)
- **2-4번 구조**: 자유 서술형 회의록, 액션아이템이 "~하겠습니다", "~해주세요", "~까지 드리겠습니다" 같은 패턴으로 산재
- **3-2번 결과 파일명**: `회의록_액션아이템_2026-04-15.xlsx`
- **3-3번 꼭 들어갈 것**: 담당자, 할 일, 마감일, 우선순위(상/중/하), 비고(원문 발췌)
- **3-4번 서식**: 마감 지난 항목 빨간 글씨, 담당자 미정은 회색
- **4-1번 규칙**:
  - 마감일이 "다음 주", "이번 달 말" 같으면 회의 날짜 기준으로 환산
  - 담당자가 명시 안 되면 "미정"
- **4-2번 예외**:
  - "검토해보겠습니다" 같이 모호한 표현도 액션아이템으로 포함하되 우선순위 "하"
  - 단순 보고 문장은 제외 ("3월 매출 120억이었습니다" → 액션 아님)
- **6번 검증**: 내가 회의록을 읽어서 센 액션아이템 개수와 결과 행 수가 일치
- **8번 오늘 목표**: 샘플 회의록 1개에서 액션아이템 엑셀 1개 완성

---

## Step 2: Claude에게 요청 (15분)

```
방금 채운 CLAUDE.md 규칙에 따라 회의록_2026-04-15.txt에서
액션아이템을 뽑아 엑셀로 정리해줘.
```

### 흔히 나오는 문제

- **"~하겠습니다"도 아닌데 액션으로 잡힘**
  ```
  단순 상황 보고("매출이 X였습니다")는 제외하고, 미래의 행동을 약속한 것만 잡아줘.
  ```
- **마감일 해석이 달라짐**
  ```
  "다음 주"는 회의 날짜(2026-04-15) 기준 다음 주 금요일(2026-04-24)로 해석해줘.
  "이번 달 말"은 2026-04-30.
  ```
- **우선순위 기준이 모호**
  ```
  우선순위 기준을 명확히 해줘:
  - 상: 마감 7일 이내 + 담당자 명시
  - 중: 마감 있음 + 담당자 명시
  - 하: 마감 모호 or 담당자 미정
  ```

---

## Step 3: 검증 (5분)

### 이 케이스만의 검증 포인트

1. **원문 회의록을 직접 읽어서** 액션아이템 몇 개인지 세본다
2. 결과 엑셀 행 수와 일치하는지 확인
3. 담당자가 명시된 항목은 원문 그 인물과 일치하는지 (Claude가 엉뚱하게 배정하지 않았나)
4. 마감일 계산이 맞나 (회의 날짜 + N일 수기 확인)

---

## 확장 (시간 남으면)

- 담당자별 시트 분리 (`담당자`별 탭)
- 우선순위 "상"만 이메일 본문으로 출력
- 다음 주 할 일을 받은 사람이 바로 보낼 수 있는 리마인더 메일 초안 생성
