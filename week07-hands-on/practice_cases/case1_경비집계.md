# Case 1 — 경비 집계

> **난이도 ★★☆ — 재경팀 연관성 ★★★**
> 실제 월마감 업무와 가장 비슷한 실습. week04에서 만든 경비 엑셀을 재활용합니다.

---

## 하려는 일

한 달치 경비 내역(15~30건)을 **부서 × 항목 피벗 표**로 집계하고, 부서별 합계·항목별 합계·총계가 들어간 정리 엑셀을 만듭니다.

### 완성 기준 (이것까지 가면 끝)

- 부서 행 × 항목 열의 피벗 표
- 각 부서별 소계
- 각 항목별 소계
- 전체 총계
- 헤더는 남색 배경 + 흰 글씨, 합계는 회색 배경 + 볼드
- 숫자는 천단위 콤마
- 결과 파일명: `경비집계_YYYY-MM.xlsx`

---

## 샘플 데이터

week04에서 이미 만든 경비 엑셀을 재사용합니다.

**경로**: `../week04-hands-on/data/경비내역.xlsx`

**구조**:
| 일자 | 부서 | 항목 | 금액 | 결제방법 |
|---|---|---|---|---|
| `2026-03-01` | `영업1팀` | `교통비` | `15000` | `법인카드` |
| ... | ... | ... | ... | ... |

데이터가 없으면 Claude에게:
```
../week04-hands-on/data/ 에 경비내역.xlsx가 있는지 확인하고,
없으면 부서 3개(영업1팀/영업2팀/관리팀) × 항목 4개(교통비/식대/사무용품/접대비)의
샘플 데이터 20건을 만들어서 내 업무 폴더로 복사해줘. 금액은 5천원~10만원 범위.
```

## Step 1: CLAUDE.md 빈칸 채우기 (5분)

`수강생과제_CLAUDE_템플릿.md`를 복사한 뒤 아래처럼 채우면 됩니다. 직접 타이핑 대신 Claude에게 시켜도 OK.

### 미리 알려진 값 (Claude에게 그대로 알려주세요)

- **0번 프로젝트 이름**: "한 달치 경비 내역을 부서×항목 피벗으로 집계"
- **2-1번 파일 형식**: 엑셀 (`.xlsx`)
- **2-3번 파일 개수**: 1개, 주기: 매월
- **2-4번 시트/컬럼**: 시트명 `Sheet`, 컬럼 = 일자/부서/항목/금액/결제방법
- **3-1번 결과 형식**: 엑셀
- **3-2번 결과 파일명**: `경비집계_2026-03.xlsx`
- **3-3번 꼭 들어갈 것**: 부서 소계, 항목 소계, 전체 총계, 작성일시
- **3-4번 서식**: 헤더 남색+흰색, 합계 회색+볼드, 천단위 콤마
- **4-1번 계산 규칙**: 원화만 사용(환율 불필요), 빈 칸은 0이 아니라 무시
- **4-2번 예외**: 결제방법이 "개인카드"인 행도 포함 (필터링 X)
- **6번 검증**: 부서별 소계의 합 = 항목별 소계의 합 = 전체 총계 (3중 일치)
- **7-1번 반복**: 매월 수동 실행
- **8번 오늘 목표**: 2026년 3월 한 달치 경비를 집계한 엑셀 1개 완성

## Step 2: Claude에게 자동화 요청 (20분)

CLAUDE.md 다 채우고 Claude에게:

```
방금 채운 CLAUDE.md 규칙에 따라 2026년 3월 경비 집계 엑셀을 만들어줘.
입력은 ../week04-hands-on/data/경비내역.xlsx, 출력은 같은 폴더에 경비집계_2026-03.xlsx.
```

결과가 나오면 파일을 열어봅니다.

### 흔히 나오는 문제 & 수정 프롬프트

- **부서 순서가 가나다순으로 나옴**
  ```
  부서 순서를 영업1팀 → 영업2팀 → 관리팀으로 고정해줘. 가나다순 아님.
  ```
- **항목 순서가 불안정**
  ```
  항목 순서를 교통비 → 식대 → 사무용품 → 접대비 고정 순서로.
  ```
- **합계 행이 중간에 있음**
  ```
  부서별 소계는 각 부서 블록 끝, 전체 총계는 맨 마지막 한 행으로.
  ```

## Step 3: 검증 (5분)

[../검증체크리스트.md](../검증체크리스트.md) 5개 전부 통과시키세요.

### 이 케이스만의 검증 포인트

1. **3중 합계 일치**: 부서별 소계 합 = 항목별 소계 합 = 전체 총계. 한 개라도 다르면 계산 버그.
2. **원본 행 수 = 피벗 내부 칸 수의 비어있지 않은 합**: 누락된 경비가 없는지.
3. **개인카드 포함되었나**: 결제방법이 "개인카드"인 행도 집계에 들어갔나. 흔히 빠짐.
