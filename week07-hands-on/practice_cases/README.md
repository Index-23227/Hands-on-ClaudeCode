# practice_cases/ — 본인 업무 없는 수강생용 공용 연습 케이스

> Step 1 워크숍에서 가져올 본인 업무가 없거나, 데이터 준비가 안 된 수강생을 위한 대체 실습.
> 각 케이스는 **30~45분 안에 끝나는 크기**로 설계되어 있습니다.
> 원하는 하나를 고르세요.

---

## 3개 중 선택

| 케이스 | 난이도 | 재경팀 연관성 | 추천 대상 |
|---|---|---|---|
| [Case 1 — 경비 집계](case1_경비집계.md) | ★★☆ | ★★★ | 재경팀 업무에 가장 가까운 걸 하고 싶은 분 |
| [Case 2 — 회의록 추출](case2_회의록추출.md) | ★☆☆ | ★☆☆ | 텍스트 다루는 걸 보고 싶은 분 |
| [Case 3 — 이메일 일괄 생성](case3_이메일일괄.md) | ★★☆ | ★★☆ | "반복 작업 한 번에" 감각을 체험하고 싶은 분 |

각 케이스는 **본인 업무 가져온 수강생의 Step 1 흐름과 동일**합니다:
1. 샘플 데이터 확인
2. CLAUDE.md 템플릿 채우기 (일부 빈칸은 이미 채워져 있음)
3. Claude에게 자동화 요청
4. 검증체크리스트 돌리기
